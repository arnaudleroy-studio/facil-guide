## Lo que vas a aprender

Vas a aprender a instalar la aplicacion Kindle en tu telefono, buscar y descargar libros, ajustar el tamano del texto para leer comodamente y usar marcadores para guardar tu progreso.

## Lo que necesitas

- Un telefono inteligente (iPhone o Android)
- Una conexion Wi-Fi o datos moviles
- Una cuenta gratuita de Amazon

### Paso 1: Instala la aplicacion Kindle

Abre la **App Store** (iPhone) o la **Google Play Store** (Android). Toca la barra de busqueda y escribe **Kindle**. Busca la aplicacion llamada **Kindle** de AMZN Mobile LLC. Toca **Obtener** o **Instalar**. Espera a que se descargue y luego toca **Abrir**.

### Paso 2: Inicia sesion con tu cuenta de Amazon

Cuando la aplicacion se abra, te pedira que inicies sesion. Escribe tu correo electronico de Amazon y tu contrasena, luego toca **Iniciar sesion**. Si no tienes una cuenta de Amazon, toca **Crear una nueva cuenta de Amazon** y sigue las instrucciones. Solo necesitas una direccion de correo electronico y una contrasena.

### Paso 3: Busca un libro para leer

Toca la pestana **Descubrir** o **Tienda** en la parte inferior de la pantalla. Puedes explorar por categoria como Misterio, Romance, Historia o Biografia. Tambien puedes tocar el icono de **Busqueda** y escribir el nombre de un libro o autor. Para encontrar libros gratis, busca "ebooks gratis" o navega por la seccion Top Gratis.

### Paso 4: Descarga un libro

Cuando encuentres un libro que te interese, toca sobre el para ver su descripcion y resenas. Toca **Comprar** para libros de pago u **Obtener** para libros gratis. El libro se descargara automaticamente en tu biblioteca Kindle. Vuelve a la pestana **Biblioteca** para ver todos tus libros descargados.

### Paso 5: Empieza a leer

Toca la portada del libro en tu Biblioteca para abrirlo. El libro aparecera en tu pantalla como una pagina real. Para pasar a la pagina siguiente, desliza el dedo de derecha a izquierda en la pantalla, o toca el lado derecho. Para volver atras, desliza de izquierda a derecha o toca el lado izquierdo.

### Paso 6: Agranda el texto

Toca el centro de la pantalla para mostrar la barra de menu. Toca el boton **Aa** en la parte superior. Veras opciones para cambiar el tamano del texto. Mueve el control deslizante para hacer el texto mas grande o mas pequeno. Tambien puedes cambiar el estilo de fuente y el color de fondo. Prueba el fondo sepia (amarillo calido) para leer mas comodamente de noche.

### Paso 7: Usa marcadores y subrayados

Para guardar tu posicion, toca la esquina superior derecha de cualquier pagina. Aparecera un pequeno icono de marcador. Para encontrar tus marcadores despues, toca el centro de la pantalla y luego el icono de **marcador** o **Ir a** en el menu. Tambien puedes mantener presionada una palabra para subrayarla o buscar su definicion.

## Consejos

- Lee con el telefono en modo vertical para la mejor experiencia
- Activa **No molestar** mientras lees para que las notificaciones no te interrumpan
- Usa un tamano de texto grande y un fondo calido para reducir la fatiga visual
- La aplicacion recuerda donde te quedaste, asi que puedes cerrarla y volver cuando quieras
- Pide libros prestados gratis de tu biblioteca local con la aplicacion Libby y leelos en Kindle
