## What you will learn

In this guide, you will learn how to install new programs (also called applications or software) on a Windows computer. We will cover two methods: using the Microsoft Store and downloading from a website.

## What you need

- A Windows computer (Windows 10 or Windows 11)
- An internet connection
- Administrator access to the computer (usually the main account)

## Step-by-step instructions

### Step 1: Decide what you want to install

Think about what you need the program for. Common examples:

- **Video calls:** Zoom, Skype, or Google Meet
- **Writing documents:** Microsoft Word or LibreOffice
- **Web browsing:** Google Chrome or Firefox
- **Photo editing:** Paint or GIMP
- **Antivirus:** Windows Defender (already built in) or Avast

### Step 2: Try the Microsoft Store first

Click the **Start** button (the Windows icon at the bottom left of your screen). Type **Microsoft Store** and click on it.

In the Store, use the search bar at the top to find the program you want. Click on the program, then click the **Get** or **Install** button.

The program will download and install automatically. When it is finished, you will see an **Open** button. You can also find it in your Start menu.

### Step 3: Download from the official website

If the program is not in the Microsoft Store, you can download it from the program's official website.

1. Open your web browser (Edge, Chrome, or Firefox).
2. Search for the program's name followed by "download" (for example, "Zoom download").
3. Go to the **official website** (check that the web address matches the company name).
4. Click the **Download** button on the website.

A file will be downloaded to your **Downloads** folder.

### Step 4: Run the installer

Open your **Downloads** folder (press **Ctrl + J** in your browser, or open File Explorer and click Downloads).

Double-click the file you just downloaded. It usually ends in **.exe**.

Windows may ask **"Do you want to allow this app to make changes to your device?"** Click **Yes** if you trust the program.

### Step 5: Follow the installation steps

An installation window will open. It usually asks you to:

1. Accept the terms and conditions (check the box and click **Next**).
2. Choose where to install the program (the default location is fine for most people, just click **Next**).
3. Click **Install** and wait for it to finish.
4. Click **Finish** or **Close**.

The program is now installed and ready to use.

### Step 6: Find and open the program

Click the **Start** button and scroll through the list of apps to find it, or type the program's name in the search bar.

You can right-click on the program and select **Pin to taskbar** to add a shortcut at the bottom of your screen for easy access.

## Tips

- Always download programs from official sources. If a website asks you to download something you did not search for, close the page.
- If you see extra offers during installation (like toolbars or other programs), uncheck those boxes. You do not need them.
- Keep your programs updated. Updates fix security problems and add new features. Most programs will notify you when an update is available.
