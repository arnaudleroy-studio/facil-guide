## Ce que vous allez apprendre

Dans ce guide, vous apprendrez a utiliser le gestionnaire de mots de passe integre a votre iPhone ou telephone Android. Un gestionnaire de mots de passe enregistre tous vos mots de passe dans un endroit sur, pour que vous n'ayez pas a retenir chacun d'entre eux. Quand vous visitez un site ou ouvrez une application, le telephone remplit votre mot de passe automatiquement.

## Ce dont vous avez besoin

- Un iPhone ou un telephone Android
- Le verrouillage de votre telephone configure avec un code PIN, une empreinte digitale ou la reconnaissance faciale
- Une connexion internet

### Etape 1 : Verifiez que votre ecran de verrouillage est configure

Le gestionnaire de mots de passe de votre telephone ne fonctionne que si votre telephone a un ecran de verrouillage. C'est le code PIN, l'empreinte digitale ou la reconnaissance faciale que vous utilisez pour deverrouiller votre telephone. Si ce n'est pas encore fait, allez dans **Reglages**, puis **Face ID et code** (sur iPhone) ou **Reglages**, puis **Securite** (sur Android). Suivez les instructions pour creer un code PIN ou configurer votre empreinte digitale.

### Etape 2 : Activez le gestionnaire de mots de passe

Sur la plupart des telephones, le gestionnaire de mots de passe est deja active. Pour verifier :

- **Sur iPhone :** Ouvrez **Reglages**, faites defiler vers le bas et appuyez sur **Mots de passe**. On vous demandera de deverrouiller avec Face ID, Touch ID ou votre code. Si vous voyez une liste de mots de passe enregistres, c'est deja en marche.
- **Sur Android :** Ouvrez **Reglages**, appuyez sur **Google**, puis sur **Gerer votre compte Google**. Allez dans l'onglet **Securite** et cherchez **Gestionnaire de mots de passe**. Vous pouvez aussi ouvrir **Chrome**, appuyer sur les trois points en haut a droite, et appuyer sur **Gestionnaire de mots de passe**.

### Etape 3 : Enregistrez un mot de passe pour la premiere fois

La prochaine fois que vous vous connectez a un site ou une application, votre telephone proposera d'enregistrer le mot de passe. Apres avoir tape votre email et votre mot de passe et appuye sur **Se connecter**, un petit message apparaitra vous demandant si vous voulez enregistrer le mot de passe. Appuyez sur **Enregistrer le mot de passe** ou **Enregistrer**. Le telephone s'en souviendra pour la prochaine fois.

### Etape 4 : Laissez le telephone remplir votre mot de passe automatiquement

Quand vous retournez sur un site ou une application ou vous avez enregistre un mot de passe, appuyez sur le champ du nom d'utilisateur ou de l'email. Votre telephone affichera une suggestion avec votre identifiant enregistre en haut du clavier ou dans une fenetre. Appuyez dessus, et le telephone remplira automatiquement votre email et votre mot de passe. Vous devrez peut-etre confirmer avec votre empreinte digitale ou votre visage.

### Etape 5 : Retrouvez et consultez vos mots de passe enregistres

Si vous avez besoin de voir un mot de passe que vous avez enregistre :

- **Sur iPhone :** Ouvrez **Reglages**, appuyez sur **Mots de passe**, et deverrouillez avec Face ID ou votre code. Vous verrez une liste de tous vos comptes enregistres. Appuyez sur n'importe quelle entree pour voir le nom d'utilisateur et le mot de passe.
- **Sur Android :** Ouvrez **Reglages**, appuyez sur **Google**, puis **Gestionnaire de mots de passe**. Ou ouvrez **Chrome**, appuyez sur les trois points, puis **Gestionnaire de mots de passe**. Appuyez sur n'importe quelle entree pour voir les details.

### Etape 6 : Laissez le telephone vous suggerer un mot de passe fort

Quand vous creez un nouveau compte sur un site ou une application, appuyez sur le champ du mot de passe. Votre telephone vous suggerera un mot de passe fort compose de caracteres aleatoires. Sur iPhone, vous verrez **Utiliser un mot de passe fort** apparaitre au-dessus du clavier. Sur Android, Chrome affichera **Suggerer un mot de passe fort**. Appuyez dessus pour utiliser la suggestion. Le telephone l'enregistrera automatiquement.

### Etape 7 : Verifiez vos mots de passe faibles ou reutilises

Votre telephone peut vous dire si certains de vos mots de passe sont faibles, reutilises ou apparus dans une fuite de donnees.

- **Sur iPhone :** Allez dans **Reglages**, puis **Mots de passe**, puis appuyez sur **Recommandations de securite** en haut. La liste des mots de passe a corriger s'affichera.
- **Sur Android :** Ouvrez le **Gestionnaire de mots de passe** (via Reglages ou Chrome) et cherchez **Verification des mots de passe**. Il analysera vos mots de passe et vous avertira des problemes.

Si vous voyez un avertissement, appuyez sur l'entree et appuyez sur **Changer le mot de passe** pour visiter le site et le mettre a jour.

## Conseils

- Utilisez un mot de passe different pour chaque compte. Le gestionnaire les retient tous, vous n'avez pas a le faire.
- Ne partagez jamais votre code PIN avec qui que ce soit. Il protege tous vos mots de passe enregistres.
- Si votre telephone vous suggere un mot de passe fort, acceptez-le toujours. Les mots de passe aleatoires sont bien plus surs que ceux que vous inventez.
- Verifiez les recommandations de securite une fois par mois pour rester protege.
