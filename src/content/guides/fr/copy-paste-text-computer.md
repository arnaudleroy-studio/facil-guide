## Ce que vous allez apprendre

Dans ce guide, vous allez apprendre a copier du texte d'un endroit et a le coller dans un autre sur votre ordinateur. C'est l'une des competences les plus utiles que vous puissiez apprendre. Cela fait gagner du temps et evite les erreurs de frappe.

## Ce dont vous avez besoin

- Un ordinateur (Windows ou Mac)
- Un clavier et une souris

## Les etapes

### Etape 1 : Selectionner le texte que vous voulez copier

Cliquez au debut du texte que vous voulez copier. Maintenez le bouton de la souris enfonce et faites glisser jusqu'a la fin du texte. Le texte selectionne sera surligne en bleu.

**Astuce :** Pour selectionner tout le texte d'une page, appuyez sur **Ctrl + A** (Windows) ou **Cmd + A** (Mac).

### Etape 2 : Copier le texte selectionne

Avec le texte surligne, appuyez sur **Ctrl + C** (Windows) ou **Cmd + C** (Mac) sur votre clavier.

Rien de visible ne se passera, mais le texte a ete enregistre dans le presse-papiers de votre ordinateur (une zone de stockage temporaire).

Vous pouvez aussi faire un clic droit sur le texte surligne et selectionner **Copier** dans le menu qui apparait.

### Etape 3 : Cliquer la ou vous voulez coller

Ouvrez le document, l'email ou la zone de texte ou vous voulez mettre le texte. Cliquez une fois pour placer votre curseur exactement a l'endroit voulu.

### Etape 4 : Coller le texte

Appuyez sur **Ctrl + V** (Windows) ou **Cmd + V** (Mac) sur votre clavier.

Le texte que vous avez copie apparaitra a la position du curseur. Vous pouvez coller le meme texte plusieurs fois en appuyant a nouveau sur le raccourci.

Vous pouvez aussi faire un clic droit et selectionner **Coller** dans le menu.

### Etape 5 : Apprendre le raccourci Couper

Si vous voulez deplacer du texte au lieu de le copier, utilisez **Couper** au lieu de **Copier** :

1. Selectionnez le texte.
2. Appuyez sur **Ctrl + X** (Windows) ou **Cmd + X** (Mac).
3. Cliquez la ou vous voulez le deplacer.
4. Appuyez sur **Ctrl + V** (Windows) ou **Cmd + V** (Mac) pour coller.

Le texte disparaitra de son emplacement d'origine et apparaitra au nouvel endroit.

## Conseils

- Ces raccourcis fonctionnent presque partout : navigateurs web, emails, traitements de texte et meme les dossiers de fichiers.
- Pour copier un fichier sur votre ordinateur, cliquez dessus une fois pour le selectionner, puis utilisez les memes raccourcis **Ctrl + C** et **Ctrl + V**.
- Si vous collez du texte et qu'il a un aspect different du reste de votre document (police ou taille differente), utilisez **Ctrl + Shift + V** (Windows) ou **Cmd + Shift + V** (Mac) pour coller sans mise en forme. Cela ne colle que le texte brut.
