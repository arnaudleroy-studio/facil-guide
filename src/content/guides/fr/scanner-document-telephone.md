## Ce que vous allez apprendre

Dans ce guide, vous allez apprendre a utiliser l'appareil photo de votre telephone pour scanner des documents papier et les transformer en fichiers numeriques. C'est utile pour garder des copies de lettres importantes, de factures, d'ordonnances ou de formulaires sans avoir besoin d'un scanner ou d'une imprimante.

## Ce dont vous avez besoin

- Un iPhone ou un telephone Android avec un appareil photo
- Le document que vous souhaitez scanner (lettre, facture, formulaire ou tout autre papier)
- Une surface plate et bien eclairee pour poser le document

### Etape 1 : Posez le document sur une surface plate

Placez le document papier a plat sur une table ou un bureau. Assurez-vous que l'endroit est bien eclaire, idealement pres d'une fenetre ou sous une lampe. Lissez les plis ou les froissures pour que le texte soit facile a lire. Evitez les zones sombres ou les ombres.

### Etape 2 : Ouvrez le scanner sur iPhone

Ouvrez l'application **Notes**. Appuyez sur le bouton **Nouvelle note** (l'icone de carre avec un stylo en bas a droite). Appuyez sur l'icone **Appareil photo** au-dessus du clavier, puis appuyez sur **Scanner des documents**. Votre appareil photo s'ouvrira en mode scanner.

### Etape 3 : Ouvrez le scanner sur Android

Ouvrez l'application **Google Drive**. Appuyez sur le bouton **+** en bas a droite. Appuyez sur **Scanner**. Votre appareil photo s'ouvrira en mode scanner. Si vous n'avez pas Google Drive, vous pouvez le telecharger gratuitement depuis le **Play Store**.

### Etape 4 : Pointez votre appareil photo vers le document

Tenez votre telephone au-dessus du document pour que toute la page apparaisse a l'ecran. Gardez le telephone parallele au papier (pas en angle). Le scanner detectera automatiquement les bords du document. Sur iPhone, il peut capturer automatiquement. Sur Android, appuyez sur le **bouton de capture** (le bouton rond) pour prendre la photo.

### Etape 5 : Ajustez les coins si necessaire

Apres la capture, vous verrez l'image scannee avec quatre poignees aux coins. Si les bords ne sont pas parfaitement alignes avec le document, faites glisser chaque **poignee de coin** pour correspondre aux bords du papier. Appuyez sur **Garder le scan** sur iPhone ou sur la **coche** sur Android.

### Etape 6 : Ajoutez d'autres pages ou terminez

Si vous avez d'autres pages a scanner, pointez votre appareil photo vers la page suivante et repetez l'operation. Le scanner ajoutera chaque page au meme document. Quand vous avez fini de scanner toutes les pages, appuyez sur **Enregistrer** sur iPhone ou sur le bouton **Enregistrer** sur Android.

### Etape 7 : Partagez ou enregistrez le document scanne

Sur iPhone, le scan est enregistre dans votre note. Appuyez sur le scan, puis appuyez sur le bouton **Partager** (le carre avec une fleche) pour l'envoyer par email ou message. Sur Android, le scan est enregistre en PDF dans votre Google Drive. Appuyez sur le fichier, puis sur le menu **trois points** et choisissez **Partager** pour l'envoyer.

## Conseils

- Nettoyez l'objectif de l'appareil photo de votre telephone avec un chiffon doux avant de scanner pour obtenir les resultats les plus nets
- Si vous scannez un petit ticket de caisse, placez-le sur un fond contrastant pour que le scanner detecte mieux les bords
- Pour les documents importants, scannez en bonne lumiere naturelle plutot que sous un eclairage artificiel faible
- Vous pouvez renommer le fichier scanne avec un nom descriptif comme "Facture electricite janvier" pour le retrouver facilement plus tard
