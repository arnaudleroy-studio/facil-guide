## Ce que vous allez apprendre

Dans ce guide, vous allez apprendre a installer de nouveaux programmes (aussi appeles applications ou logiciels) sur un ordinateur Windows. Nous couvrirons deux methodes : le Microsoft Store et le telechargement depuis un site web.

## Ce dont vous avez besoin

- Un ordinateur Windows (Windows 10 ou Windows 11)
- Une connexion internet
- Un acces administrateur a l'ordinateur (generalement le compte principal)

## Les etapes

### Etape 1 : Decider ce que vous voulez installer

Reflechissez a ce dont vous avez besoin. Exemples courants :

- **Appels video :** Zoom, Skype ou Google Meet
- **Redaction de documents :** Microsoft Word ou LibreOffice
- **Navigation web :** Google Chrome ou Firefox
- **Retouche photo :** Paint ou GIMP
- **Antivirus :** Windows Defender (deja integre) ou Avast

### Etape 2 : Essayer d'abord le Microsoft Store

Cliquez sur le bouton **Demarrer** (l'icone Windows en bas a gauche de votre ecran). Tapez **Microsoft Store** et cliquez dessus.

Dans le Store, utilisez la barre de recherche en haut pour trouver le programme souhaite. Cliquez sur le programme, puis sur le bouton **Obtenir** ou **Installer**.

Le programme se telechargera et s'installera automatiquement. Une fois termine, vous verrez un bouton **Ouvrir**. Vous le trouverez aussi dans votre menu Demarrer.

### Etape 3 : Telecharger depuis le site officiel

Si le programme n'est pas dans le Microsoft Store, vous pouvez le telecharger depuis son site officiel.

1. Ouvrez votre navigateur web (Edge, Chrome ou Firefox).
2. Cherchez le nom du programme suivi de "telecharger" (par exemple, "Zoom telecharger").
3. Allez sur le **site officiel** (verifiez que l'adresse web correspond au nom de l'entreprise).
4. Cliquez sur le bouton **Telecharger** du site.

Un fichier sera telecharge dans votre dossier **Telechargements**.

### Etape 4 : Lancer l'installateur

Ouvrez votre dossier **Telechargements** (appuyez sur **Ctrl + J** dans votre navigateur, ou ouvrez l'Explorateur de fichiers et cliquez sur Telechargements).

Double-cliquez sur le fichier que vous venez de telecharger. Il se termine generalement par **.exe**.

Windows peut demander **"Voulez-vous autoriser cette application a apporter des modifications a votre appareil ?"** Cliquez sur **Oui** si vous faites confiance au programme.

### Etape 5 : Suivre les etapes d'installation

Une fenetre d'installation s'ouvrira. Elle demande generalement de :

1. Accepter les conditions d'utilisation (cochez la case et cliquez sur **Suivant**).
2. Choisir ou installer le programme (l'emplacement par defaut convient a la plupart des gens, cliquez simplement sur **Suivant**).
3. Cliquer sur **Installer** et attendre la fin.
4. Cliquer sur **Terminer** ou **Fermer**.

Le programme est maintenant installe et pret a l'emploi.

### Etape 6 : Trouver et ouvrir le programme

Cliquez sur le bouton **Demarrer** et parcourez la liste des applications pour le trouver, ou tapez son nom dans la barre de recherche.

Vous pouvez faire un clic droit sur le programme et selectionner **Epingler a la barre des taches** pour ajouter un raccourci en bas de votre ecran.

## Conseils

- Telechargez toujours les programmes depuis des sources officielles. Si un site vous demande de telecharger quelque chose que vous n'avez pas cherche, fermez la page.
- Si vous voyez des offres supplementaires pendant l'installation (comme des barres d'outils ou d'autres programmes), decochez ces cases. Vous n'en avez pas besoin.
- Gardez vos programmes a jour. Les mises a jour corrigent des problemes de securite et ajoutent de nouvelles fonctionnalites. La plupart des programmes vous prviendront quand une mise a jour est disponible.
