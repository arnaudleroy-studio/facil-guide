## Ce que vous allez apprendre

Dans ce guide, vous apprendrez a connecter votre telephone a votre ordinateur avec un cable USB et a copier les photos de votre telephone vers votre ordinateur. C'est utile pour sauvegarder vos photos, liberer de l'espace sur votre telephone ou voir vos photos sur un ecran plus grand.

## Ce dont vous avez besoin

- Votre telephone (iPhone ou Android)
- Un ordinateur (PC Windows ou Mac)
- Le cable USB fourni avec votre telephone (le meme que celui pour le charger)

## Etapes

### Etape 1 : Connecter votre telephone a votre ordinateur

Prenez le cable USB que vous utilisez pour charger votre telephone. Branchez la petite extremite dans votre telephone et la grande extremite USB dans un port USB de votre ordinateur. Votre ordinateur peut prendre quelques secondes pour reconnaitre votre telephone.

### Etape 2 : Deverrouiller votre telephone et autoriser l'acces

Quand vous branchez le cable, votre telephone peut afficher un message. Sur un iPhone, une fenetre demandera **"Faire confiance a cet ordinateur ?"** -- appuyez sur **Se fier** et entrez votre code. Sur un telephone Android, une notification peut apparaitre demandant de choisir un type de connexion. Appuyez sur la notification et selectionnez **Transfert de fichiers**.

### Etape 3 : Ouvrir l'outil de transfert de photos sur Windows

Sur un PC Windows, l'application **Photos** peut s'ouvrir automatiquement. Sinon, cliquez sur le bouton **Demarrer**, tapez **Photos** et ouvrez l'application. Cliquez sur **Importer** en haut, puis selectionnez **A partir d'un appareil connecte**. Votre ordinateur analysera votre telephone pour trouver les photos.

### Etape 4 : Ouvrir l'outil de transfert de photos sur Mac

Sur un Mac, l'application **Photos** s'ouvrira generalement automatiquement quand vous connectez un iPhone. Sinon, ouvrez **Photos** depuis le dock ou le dossier Applications. Votre telephone apparaitra dans la barre laterale gauche. Cliquez dessus pour voir vos photos. Pour un telephone Android sur Mac, telechargez et ouvrez **Android File Transfer** depuis internet, puis naviguez vers le dossier DCIM.

### Etape 5 : Selectionner les photos que vous voulez transferer

Parcourez les photos affichees sur l'ecran de votre ordinateur. Cliquez sur des photos individuelles pour les selectionner, ou cliquez sur **Tout selectionner** pour tout choisir. Si vous ne voulez que les photos recentes, cherchez un filtre par date ou triez par date.

### Etape 6 : Copier les photos vers votre ordinateur

Cliquez sur **Importer** ou **Importer la selection** pour commencer a copier les photos. Choisissez un dossier ou les sauvegarder, comme **Images** ou **Bureau**. Attendez la fin du transfert. Une barre de progression vous montrera l'avancement. Ne debranchez pas le cable tant que ce n'est pas fini.

### Etape 7 : Verifier et deconnecter

Apres la fin du transfert, ouvrez le dossier sur votre ordinateur ou vous avez sauvegarde les photos. Verifiez que les photos sont la et semblent correctes. Une fois que vous avez confirme que tout a ete transfere correctement, deconnectez votre telephone en debranchant le cable.

## Conseils

- Creez un dossier sur votre ordinateur avec la date du jour, comme "Photos 2026-02-25," pour les retrouver facilement plus tard.
- Transferez vos photos regulierement, par exemple une fois par mois, pour toujours avoir une sauvegarde.
- Si votre ordinateur a des ports USB-A (les grands rectangulaires) et USB-C (les petits ovales), utilisez celui qui correspond a votre cable. Vous pouvez acheter un adaptateur si necessaire.
- Apres avoir confirme que les photos sont bien sur votre ordinateur, vous pouvez les supprimer de votre telephone pour liberer de l'espace.
