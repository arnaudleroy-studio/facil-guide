## Ce que vous allez apprendre

Dans ce guide, vous allez apprendre a connecter votre ordinateur a un reseau Wi-Fi. Une fois connecte, vous pourrez naviguer sur internet, envoyer des emails, passer des appels video et bien plus.

## Ce dont vous avez besoin

- Un ordinateur avec le Wi-Fi (la plupart des portables l'ont integre)
- Le nom de votre reseau Wi-Fi et son mot de passe (fournis par votre box ou operateur internet)

## Les etapes

### Etape 1 : Verifier que le Wi-Fi est active

**Sur Windows :** Regardez en bas a droite de votre ecran (la barre des taches). Cliquez sur l'**icone Wi-Fi** (elle ressemble a un petit eventail ou des ondes radio). Si le Wi-Fi est desactive, cliquez sur le bouton **Wi-Fi** pour l'activer.

**Sur Mac :** Regardez en haut a droite de votre ecran. Cliquez sur l'**icone Wi-Fi** (en forme d'eventail). Si le Wi-Fi est desactive, cliquez sur **Activer le Wi-Fi**.

### Etape 2 : Voir les reseaux disponibles

Une fois le Wi-Fi active, votre ordinateur cherchera les reseaux a proximite.

**Sur Windows :** Cliquez sur l'icone Wi-Fi en bas a droite. Une liste des reseaux disponibles apparaitra.

**Sur Mac :** Cliquez sur l'icone Wi-Fi en haut a droite. Une liste deroulante des reseaux disponibles apparaitra.

Vous devriez voir le nom de votre reseau Wi-Fi domestique dans cette liste.

### Etape 3 : Selectionner votre reseau

Cliquez sur le nom de votre reseau Wi-Fi. Si vous etes chez vous, c'est generalement le nom imprime sur votre box internet.

Cochez la case **Se connecter automatiquement** si vous voulez que votre ordinateur se reconnecte a ce reseau a l'avenir sans le demander.

Cliquez sur **Connecter**.

### Etape 4 : Entrer le mot de passe

Une zone de texte apparaitra demandant le mot de passe du reseau (aussi appele cle de securite).

Tapez votre mot de passe Wi-Fi avec soin. Les majuscules et minuscules comptent.

Cliquez sur **Suivant** (Windows) ou **Rejoindre** (Mac).

### Etape 5 : Verifier la connexion

Si le mot de passe est correct, votre ordinateur se connectera au reseau. L'icone Wi-Fi changera pour indiquer la force du signal (barres pleines = bonne connexion).

Pour tester, ouvrez votre navigateur web et allez sur n'importe quel site. Si la page se charge, vous etes connecte.

## Conseils

- Si vous vous trompez de mot de passe, votre ordinateur indiquera que la connexion a echoue. Cliquez a nouveau sur le reseau et retapez le mot de passe soigneusement.
- Si votre connexion se coupe regulierement, rapprochez-vous de la box ou retirez les obstacles entre vous et la box.
- Le Wi-Fi public (dans les cafes, aeroports, hotels) demande generalement d'accepter des conditions sur une page web qui s'ouvre automatiquement. Soyez prudent avec vos informations personnelles sur les reseaux publics.
