## Ce que vous allez apprendre

Vous allez apprendre plusieurs facons d'agrandir le clavier de votre telephone et de le rendre plus facile a utiliser. Vous apprendrez a ajuster les parametres d'affichage, a utiliser le mode paysage et a modifier les parametres du clavier.

## Ce dont vous avez besoin

- Un smartphone (iPhone ou Android)

### Etape 1 : Essayez le mode paysage pour un clavier plus large

Tournez votre telephone sur le cote pour que l'ecran soit horizontal. Le clavier deviendra automatiquement plus large, avec des touches plus grandes. Si l'ecran ne tourne pas, ouvrez les **Reglages**, cherchez **Affichage** ou **Accessibilite** et verifiez que la **rotation automatique** est activee.

### Etape 2 : Augmentez le zoom d'affichage sur iPhone

Ouvrez l'application **Reglages**. Appuyez sur **Luminosite et affichage**. Appuyez sur **Zoom de l'ecran** (en bas). Choisissez **Texte plus grand** et appuyez sur **OK**. Votre iPhone redemarrera et tout, y compris le clavier, apparaitra plus grand.

### Etape 3 : Augmentez la taille d'affichage sur Android

Ouvrez l'application **Parametres**. Appuyez sur **Affichage**. Cherchez **Taille d'affichage**, **Zoom de l'ecran** ou **Taille de l'ecran**. Deplacez le curseur vers la droite pour tout agrandir, y compris le clavier. L'apercu en haut vous montre le resultat.

### Etape 4 : Ajustez la hauteur du clavier sur Android

Beaucoup de telephones Android permettent de redimensionner le clavier directement. Ouvrez une application ou vous pouvez taper pour faire apparaitre le clavier. Cherchez une **icone d'engrenage** sur le clavier. Appuyez dessus et cherchez **Taille du clavier** ou **Hauteur du clavier**. Faites glisser les bords du clavier pour l'agrandir.

### Etape 5 : Utilisez le mode a une main pour une meilleure portee

Si le clavier est trop large et que votre pouce n'atteint pas toutes les touches, essayez le mode a une main. Sur la plupart des claviers, appuyez longuement sur la **touche virgule** ou l'**icone du globe** en bas du clavier. Vous verrez une option pour reduire le clavier a gauche ou a droite de l'ecran.

### Etape 6 : Essayez la saisie vocale

Si taper sur un petit clavier reste difficile, utilisez la saisie vocale. Appuyez sur l'**icone du microphone** sur le clavier. Parlez clairement et votre telephone tapera le texte pour vous. Appuyez a nouveau sur le microphone pour arreter.

## Conseils

- Le mode paysage offre le clavier le plus large mais ne fonctionne pas dans toutes les applications
- Le zoom d'affichage agrandit tout, pas seulement le clavier, ce qui aide aussi pour la lecture
- Entrainez-vous a taper avec l'index si vos pouces sont trop larges pour les touches
- La saisie vocale fonctionne bien pour les messages mais verifiez les erreurs avant d'envoyer
