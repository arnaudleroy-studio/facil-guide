## Ce que vous allez apprendre

Dans ce guide, vous apprendrez à configurer un verrouillage d'écran sur votre téléphone pour protéger vos informations personnelles. Un verrouillage d'écran signifie que lorsque l'écran de votre téléphone s'éteint, personne ne peut utiliser votre téléphone sans entrer un code PIN, dessiner un schéma ou utiliser son empreinte digitale. Cela protège vos photos, messages, applications bancaires et données personnelles en cas de perte ou de vol.

## Ce dont vous avez besoin

- Un iPhone ou un téléphone Android
- Quelques minutes

## Les étapes

### Étape 1 : Ouvrir les paramètres de sécurité

**Sur iPhone :** Allez dans **Réglages** et appuyez sur **Face ID et code** (sur les iPhone récents) ou **Touch ID et code** (sur les iPhone plus anciens avec un bouton d'accueil).

**Sur Android :** Allez dans **Paramètres** et appuyez sur **Sécurité** (ou **Écran de verrouillage**, ou **Sécurité et confidentialité**, selon la marque de votre téléphone). Puis appuyez sur **Verrouillage de l'écran**.

### Étape 2 : Choisir votre type de verrouillage

Vous verrez plusieurs options pour verrouiller votre téléphone :

- **Code PIN** -- un code numérique, généralement 4 ou 6 chiffres. C'est l'option la plus simple et la plus courante.
- **Mot de passe** -- un code plus long utilisant des lettres et des chiffres. Plus sécurisé mais plus difficile à taper à chaque fois.
- **Schéma** -- dessinez une forme en reliant des points sur l'écran (Android uniquement).
- **Empreinte digitale** -- utilisez votre doigt pour déverrouiller (disponible sur la plupart des téléphones modernes).
- **Reconnaissance faciale** -- utilisez votre visage pour déverrouiller (iPhone Face ID et certains téléphones Android).

Pour la plupart des gens, un **code PIN à 6 chiffres** combiné avec l'**empreinte digitale** est le meilleur équilibre entre sécurité et praticité.

### Étape 3 : Configurer un code PIN

Choisissez **Code PIN** et tapez un numéro facile à retenir pour vous mais difficile à deviner pour les autres. Évitez les numéros évidents comme 123456, votre année de naissance ou 000000. Tapez le code PIN, puis tapez-le à nouveau pour confirmer. **Notez ce code PIN** sur un papier et gardez-le dans un endroit sûr chez vous.

### Étape 4 : Configurer le déverrouillage par empreinte digitale (optionnel mais recommandé)

Configurer votre empreinte digitale signifie que vous pouvez déverrouiller votre téléphone en touchant le capteur au lieu de taper votre code PIN à chaque fois.

**Sur iPhone :** Allez dans **Réglages** > **Touch ID et code**. Appuyez sur **Ajouter une empreinte**. Placez votre doigt sur le bouton d'accueil (ou le bouton latéral sur les modèles récents) et soulevez-le plusieurs fois. Le téléphone capturera votre empreinte. Suivez les instructions à l'écran jusqu'à la fin.

**Sur Android :** Allez dans **Paramètres** > **Sécurité** > **Empreinte digitale** (ou **Biométrie**). Appuyez sur **Ajouter une empreinte** et suivez les instructions. Vous devrez placer votre doigt sur le capteur plusieurs fois sous différents angles.

### Étape 5 : Définir le délai de verrouillage

Vous pouvez choisir combien de temps votre téléphone attend avant de se verrouiller après que l'écran s'éteint.

**Sur iPhone :** Allez dans **Réglages** > **Luminosité et affichage** > **Verrouillage automatique**. Choisissez une durée comme **1 minute** ou **2 minutes**. Puis dans **Face ID et code**, assurez-vous que **Exiger le code** est réglé sur **Immédiatement**.

**Sur Android :** Allez dans **Paramètres** > **Sécurité** > paramètres du **Verrouillage de l'écran**. Cherchez **Verrouiller après la mise en veille** et choisissez **Immédiatement** ou **5 secondes** pour la meilleure sécurité.

### Étape 6 : Tester votre écran de verrouillage

Appuyez sur le **bouton d'alimentation** sur le côté de votre téléphone pour éteindre l'écran. Attendez quelques secondes, puis appuyez à nouveau pour réveiller le téléphone. Vous devriez maintenant voir un écran de verrouillage demandant votre code PIN, empreinte ou visage. Entrez votre code PIN ou utilisez votre empreinte pour déverrouiller. Si ça fonctionne, votre téléphone est maintenant protégé.

## Conseils

- Choisissez un code PIN que vous pouvez retenir facilement mais qui n'est pas évident pour les autres. Une bonne méthode est d'utiliser une date significative que vous seul connaissez.
- Enregistrez le doigt que vous utilisez le plus souvent (généralement le pouce droit ou l'index) pour le déverrouillage par empreinte.
- Si votre téléphone le permet, enregistrez deux empreintes au cas où un doigt serait mouillé ou aurait un pansement.
- Ne partagez jamais votre code PIN avec des inconnus. Partagez-le uniquement avec un membre de confiance de votre famille en cas d'urgence.
