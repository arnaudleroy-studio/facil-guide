## Ce que vous allez apprendre

Dans ce guide, vous allez apprendre a retrouver des fichiers sur votre ordinateur quand vous avez oublie ou vous les avez enregistres. Vous apprendrez a utiliser la fonction de recherche et a parcourir les dossiers courants.

## Ce dont vous avez besoin

- Un ordinateur (Windows ou Mac)

## Les etapes

### Etape 1 : Verifier d'abord les dossiers courants

La plupart des fichiers se retrouvent dans l'un de ces dossiers standard :

- **Documents** — courriers, tableurs, fichiers texte
- **Telechargements** — tout ce qui a ete telecharge depuis internet
- **Bureau** — fichiers enregistres directement sur le fond d'ecran
- **Images** — photos et images

**Sur Windows :** Ouvrez l'**Explorateur de fichiers** (cliquez sur l'icone de dossier dans la barre des taches). Les dossiers courants apparaissent sur le cote gauche.

**Sur Mac :** Ouvrez le **Finder** (l'icone de visage souriant dans le dock). Les dossiers courants apparaissent dans la barre laterale.

### Etape 2 : Utiliser la fonction de recherche sur Windows

Cliquez sur la **barre de recherche** en bas de l'ecran (ou appuyez sur la **touche Windows** et commencez a taper).

Tapez le nom du fichier, ou meme juste une partie du nom. Windows affichera les resultats correspondants de tout votre ordinateur.

Pour une recherche plus ciblee, ouvrez l'**Explorateur de fichiers** et utilisez la barre de recherche en haut a droite. Cela ne cherche que dans le dossier actuel et ses sous-dossiers.

### Etape 3 : Utiliser la fonction de recherche sur Mac

Appuyez sur **Cmd + Espace** pour ouvrir **Spotlight**, une barre de recherche au centre de votre ecran.

Tapez le nom du fichier ou un mot-cle. Spotlight affiche les resultats instantanement, y compris documents, photos, emails et applications.

Pour une recherche plus detaillee, ouvrez le **Finder** et appuyez sur **Cmd + F**. Vous pouvez filtrer par type de fichier, date de modification et d'autres criteres.

### Etape 4 : Trier les fichiers par date

Si vous vous souvenez approximativement de la derniere fois que vous avez travaille sur le fichier, triez par date :

**Sur Windows :** Ouvrez l'Explorateur de fichiers, naviguez vers un dossier et cliquez sur l'en-tete de colonne **Date de modification** pour trier du plus recent au plus ancien.

**Sur Mac :** Dans le Finder, cliquez sur **Presentation > par liste**, puis cliquez sur la colonne **Date de modification** pour trier.

Le fichier sur lequel vous avez travaille le plus recemment apparaitra en haut.

### Etape 5 : Chercher par type de fichier

Si vous connaissez le type de fichier que vous cherchez, recherchez par extension :

- **.docx** — documents Word
- **.pdf** — fichiers PDF
- **.jpg** ou **.png** — photos et images
- **.xlsx** — tableurs Excel
- **.mp4** — videos

Tapez l'extension dans la barre de recherche (par exemple, **.pdf**) pour voir tous les fichiers de ce type.

## Conseils

- Prenez l'habitude de donner des noms descriptifs a vos fichiers quand vous les enregistrez. "Declaration impots 2025" est bien plus facile a trouver que "Document1."
- Creez une structure de dossiers simple. Par exemple, un dossier principal "Personnel" avec des sous-dossiers pour "Medical", "Finances" et "Famille."
- Si vous perdez souvent des fichiers, envisagez de tout garder dans un dossier principal comme Documents et d'utiliser la recherche pour trouver ce dont vous avez besoin.
- Sur Mac, vous pouvez utiliser les **Tags** (points colores) pour marquer les fichiers importants et y acceder rapidement.
