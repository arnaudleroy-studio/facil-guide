## Ce que vous allez apprendre

Dans ce guide, vous allez apprendre a enregistrer des fichiers sur votre ordinateur. Enregistrer garantit que votre travail n'est pas perdu quand vous fermez un programme ou eteignez votre ordinateur.

## Ce dont vous avez besoin

- Un ordinateur (Windows ou Mac)
- N'importe quel programme ou vous creez ou modifiez des fichiers (editeur de texte, traitement de texte, editeur photo, etc.)

## Les etapes

### Etape 1 : Comprendre ce que signifie enregistrer

Quand vous travaillez sur un document, une photo ou tout autre fichier, vos modifications n'existent que dans la memoire temporaire de l'ordinateur. Si vous fermez le programme ou si le courant est coupe, ces modifications disparaissent.

Enregistrer ecrit votre travail sur le stockage de l'ordinateur (le disque dur ou le SSD), ou il reste de maniere permanente jusqu'a ce que vous le supprimiez.

### Etape 2 : Enregistrer pour la premiere fois

Quand vous creez un nouveau fichier, vous devez l'enregistrer une premiere fois.

Appuyez sur **Ctrl + S** (Windows) ou **Cmd + S** (Mac). Une fenetre apparaitra vous demandant de :

1. **Choisir un nom** pour le fichier. Choisissez quelque chose de descriptif pour le retrouver facilement (par exemple, "Lettre au medecin" plutot que "Document1").
2. **Choisir un emplacement** ou le fichier sera enregistre. Le dossier **Documents** est un bon choix.
3. Cliquer sur **Enregistrer**.

### Etape 3 : Enregistrer les modifications d'un fichier existant

Apres le premier enregistrement, appuyez regulierement sur **Ctrl + S** (Windows) ou **Cmd + S** (Mac) pendant que vous travaillez. Cela met a jour le fichier avec vos dernieres modifications.

Il n'y a pas de fenetre cette fois. Le fichier est enregistre instantanement au meme emplacement avec le meme nom.

**Bonne habitude :** Enregistrez toutes les quelques minutes en travaillant. Cela prend moins d'une seconde et vous protege contre la perte de travail.

### Etape 4 : Utiliser Enregistrer sous pour une copie

Si vous voulez sauvegarder une version differente d'un fichier (par exemple, une traduction ou une version mise a jour), utilisez **Enregistrer sous** :

Appuyez sur **Ctrl + Shift + S** (Windows) ou **Cmd + Shift + S** (Mac), ou allez dans **Fichier > Enregistrer sous** dans le menu.

Donnez un nouveau nom au fichier (par exemple, "Lettre au medecin - mise a jour") et choisissez un emplacement. Cliquez sur **Enregistrer**.

Le fichier original reste inchange. Vous avez maintenant deux fichiers.

### Etape 5 : Retrouver vos fichiers enregistres

Pour retrouver les fichiers que vous avez enregistres :

**Sur Windows :** Ouvrez l'**Explorateur de fichiers** (l'icone de dossier dans la barre des taches). Cliquez sur **Documents**, **Bureau** ou le dossier ou vous avez enregistre.

**Sur Mac :** Ouvrez le **Finder** (l'icone de visage souriant dans le dock). Cliquez sur **Documents**, **Bureau** ou le dossier concerne.

Vous pouvez aussi utiliser la barre de recherche. Appuyez sur la **touche Windows** et tapez le nom du fichier, ou appuyez sur **Cmd + Espace** sur Mac et tapez le nom.

## Conseils

- Nommez vos fichiers clairement. "Budget mars 2026" est bien mieux que "Sans titre" ou "Nouveau document."
- Creez des dossiers pour organiser les fichiers lies. Faites un clic droit dans un dossier et selectionnez **Nouveau > Dossier** (Windows) ou **Nouveau dossier** (Mac).
- Si un programme plante avant que vous n'ayez enregistre, cherchez les fichiers recuperes automatiquement. Beaucoup de programmes comme Word et Google Docs enregistrent des brouillons automatiquement.
