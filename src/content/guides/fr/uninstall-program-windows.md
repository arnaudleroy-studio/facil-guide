## Ce que vous allez apprendre

Dans ce guide, vous allez apprendre a supprimer les programmes dont vous n'avez plus besoin de votre ordinateur Windows. Supprimer les programmes inutilises libere de l'espace de stockage et peut aider votre ordinateur a fonctionner plus vite.

## Ce dont vous avez besoin

- Un ordinateur Windows (Windows 10 ou Windows 11)
- Un acces administrateur (generalement le compte principal)

## Les etapes

### Etape 1 : Ouvrir les Parametres

Cliquez sur le bouton **Demarrer** (l'icone Windows en bas a gauche de votre ecran). Puis cliquez sur l'**icone d'engrenage** pour ouvrir les **Parametres**.

Vous pouvez aussi appuyer sur **Touche Windows + I** pour ouvrir les Parametres directement.

### Etape 2 : Aller a la liste des applications

Dans les Parametres, cliquez sur **Applications**, puis sur **Applications installees** (sur Windows 11) ou **Applications et fonctionnalites** (sur Windows 10).

Vous verrez une liste de tous les programmes installes sur votre ordinateur, ainsi que l'espace qu'ils occupent.

### Etape 3 : Trouver et desinstaller le programme

Parcourez la liste ou utilisez la barre de recherche en haut pour trouver le programme a supprimer.

**Sur Windows 11 :** Cliquez sur les trois points (**...**) a cote du nom du programme, puis cliquez sur **Desinstaller**.

**Sur Windows 10 :** Cliquez sur le nom du programme, puis sur le bouton **Desinstaller** qui apparait.

Un message de confirmation demandera si vous etes sur. Cliquez a nouveau sur **Desinstaller**.

Certains programmes ouvriront leur propre assistant de desinstallation. Suivez les instructions a l'ecran, generalement en cliquant sur **Suivant** puis **Terminer**.

### Etape 4 : Redemarrer votre ordinateur

Apres avoir desinstalle un programme, il est recommande de redemarrer votre ordinateur. Cela garantit que tous les fichiers restants sont nettoyes correctement.

Cliquez sur **Demarrer** > **Marche/Arret** > **Redemarrer**.

## Conseils

- Triez la liste des applications par taille pour voir quels programmes occupent le plus d'espace. Cela vous aide a decider quoi supprimer en premier.
- Si vous avez installe un programme depuis le Microsoft Store, vous pouvez aussi le desinstaller en faisant un clic droit sur son icone dans le menu Demarrer et en selectionnant **Desinstaller**.
- Ne desinstallez pas votre antivirus ou navigateur web sauf si vous avez une alternative prete.
- Si un programme n'apparait pas dans la liste, c'est peut-etre une application integree a Windows. Certaines peuvent etre supprimees via un clic droit dans le menu Demarrer, mais certaines applications systeme ne peuvent pas etre desinstallees.
