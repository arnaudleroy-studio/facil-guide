## Ce que vous allez apprendre

Dans ce guide, vous apprendrez quoi faire quand vous oubliez votre mot de passe. Que ce soit pour votre e-mail, une application comme WhatsApp ou Facebook, ou votre téléphone lui-même, il y a presque toujours un moyen de retrouver l'accès.

## Ce dont vous avez besoin

- Accès au numéro de téléphone ou à l'adresse e-mail liée à votre compte
- Un ordinateur, un téléphone ou une tablette avec accès à Internet

## Les étapes

### Étape 1 : Ne paniquez pas

Oublier un mot de passe arrive à tout le monde. Votre compte et toutes vos informations sont toujours en sécurité. Vous avez juste besoin de prouver qui vous êtes pour obtenir un nouveau mot de passe.

### Étape 2 : Aller sur la page de connexion

Ouvrez l'application ou le site web où vous avez oublié votre mot de passe. Sur l'écran de connexion, cherchez un lien qui dit **"Mot de passe oublié ?"**, **"Réinitialiser le mot de passe"** ou **"Besoin d'aide pour vous connecter ?"**. Appuyez ou cliquez dessus.

### Étape 3 : Entrer votre adresse e-mail ou numéro de téléphone

Le service vous demandera de taper votre **adresse e-mail** ou votre **numéro de téléphone** lié à votre compte. Tapez-le soigneusement et appuyez sur **"Suivant"** ou **"Continuer"**.

### Étape 4 : Chercher le code de vérification

Le service vous enverra un **code de vérification**. C'est un nombre court (généralement 6 chiffres) envoyé sur votre téléphone par SMS ou à votre adresse e-mail. Ouvrez vos messages ou votre e-mail pour le trouver. Le code arrive généralement en moins d'une minute.

### Étape 5 : Entrer le code et créer un nouveau mot de passe

Retournez sur la page de réinitialisation du mot de passe et tapez le **code de vérification** que vous avez reçu. Ensuite, on vous demandera de créer un **nouveau mot de passe**. Choisissez un mot de passe d'au moins 8 caractères qui mélange des lettres et des chiffres.

**Notez votre nouveau mot de passe** sur un papier et gardez-le dans un endroit sûr chez vous.

### Étape 6 : Se connecter avec votre nouveau mot de passe

Retournez sur la page de connexion et connectez-vous avec votre **adresse e-mail** et votre **nouveau mot de passe**. Vous avez à nouveau accès à votre compte.

Si vous utilisez le même mot de passe sur d'autres services, pensez à le mettre à jour là aussi.

## Conseils utiles

- **Notez vos mots de passe** : Gardez un petit carnet chez vous avec vos mots de passe. C'est la manière la plus sûre pour la plupart des gens de s'en souvenir.
- **Utilisez des mots de passe différents** : Essayez de ne pas utiliser le même mot de passe partout. Si un compte est compromis, les autres restent protégés.
- **E-mail de secours** : Quand vous créez un compte, ajoutez toujours une adresse e-mail de récupération ou un numéro de téléphone. C'est ce qui vous permet de réinitialiser votre mot de passe plus tard.
- **Services courants** : Voici les pages de réinitialisation des mots de passe pour les services populaires :
  - Gmail : [accounts.google.com](https://accounts.google.com)
  - Facebook : [facebook.com/login/identify](https://www.facebook.com/login/identify)
  - WhatsApp : WhatsApp utilise votre numéro de téléphone et un code de vérification envoyé par SMS
  - Apple ID : [iforgot.apple.com](https://iforgot.apple.com)
