## Cosa imparerai

In questa guida, imparerai a trovare file sul computer quando hai dimenticato dove li hai salvati. Imparerai a usare la funzione di ricerca e a esplorare le cartelle piu comuni.

## Cosa ti serve

- Un computer (Windows o Mac)

## Istruzioni passo per passo

### Passo 1: Controllare prima le cartelle comuni

La maggior parte dei file finisce in una di queste cartelle standard:

- **Documenti** — lettere, fogli di calcolo, file di testo
- **Download** — tutto cio che e stato scaricato da internet
- **Desktop** — file salvati direttamente sullo sfondo del desktop
- **Immagini** — foto e immagini

**Su Windows:** Apri **Esplora File** (clicca sull'icona della cartella nella barra delle applicazioni). Le cartelle comuni appaiono sul lato sinistro.

**Su Mac:** Apri il **Finder** (l'icona della faccina sorridente nel dock). Le cartelle comuni appaiono nella barra laterale.

### Passo 2: Usare la funzione di ricerca su Windows

Clicca sulla **barra di ricerca** nella parte inferiore dello schermo (o premi il **tasto Windows** e inizia a digitare).

Digita il nome del file, o anche solo parte del nome. Windows mostrera i risultati da tutto il computer.

Per una ricerca piu mirata, apri **Esplora File** e usa la barra di ricerca nell'angolo in alto a destra. Questo cerca solo nella cartella corrente e nelle sue sottocartelle.

### Passo 3: Usare la funzione di ricerca su Mac

Premi **Cmd + Spazio** per aprire **Spotlight**, una barra di ricerca al centro dello schermo.

Digita il nome del file o una parola chiave. Spotlight mostra i risultati istantaneamente, inclusi documenti, foto, email e applicazioni.

Per una ricerca piu dettagliata, apri il **Finder** e premi **Cmd + F**. Puoi filtrare per tipo di file, data di modifica e altri criteri.

### Passo 4: Ordinare i file per data

Se ricordi approssimativamente quando hai lavorato sul file l'ultima volta, ordina per data:

**Su Windows:** Apri Esplora File, vai in una cartella e clicca sull'intestazione della colonna **Data di modifica** per ordinare dal piu recente o dal piu vecchio.

**Su Mac:** Nel Finder, clicca su **Vista > come elenco**, poi sulla colonna **Data di modifica** per ordinare.

Il file su cui hai lavorato piu di recente apparira in cima.

### Passo 5: Cercare per tipo di file

Se sai il tipo di file che cerchi, cerca per estensione:

- **.docx** — documenti Word
- **.pdf** — file PDF
- **.jpg** o **.png** — foto e immagini
- **.xlsx** — fogli di calcolo Excel
- **.mp4** — video

Digita l'estensione nella barra di ricerca (per esempio, **.pdf**) per vedere tutti i file di quel tipo.

## Consigli

- Prendi l'abitudine di dare nomi descrittivi ai file quando li salvi. "Dichiarazione tasse 2025" e molto piu facile da trovare di "Documento1."
- Crea una struttura di cartelle semplice. Per esempio, una cartella principale "Personale" con sottocartelle per "Medico", "Finanze" e "Famiglia."
- Se perdi spesso i file, considera di tenere tutto in una cartella principale come Documenti e usare la ricerca per trovare cio che ti serve.
- Su Mac, puoi usare le **Etichette** (punti colorati) per contrassegnare i file importanti e accedervi rapidamente.
