## Cosa imparerai

In questa guida, imparerai a usare una chiavetta USB (chiamata anche pen drive o memoria USB) per copiare file tra computer, creare backup o portare con te documenti importanti.

## Cosa ti serve

- Un computer (Windows o Mac) con una porta USB
- Una chiavetta USB

## Istruzioni passo per passo

### Passo 1: Collegare la chiavetta USB

Trova una porta USB sul computer. Sui portatili, le porte USB sono di solito sui lati. Sui computer fissi, sono davanti o dietro.

Inserisci la chiavetta USB nella porta. Entra solo in un verso. Se non scivola dentro facilmente, girala e riprova. Non forzare.

### Passo 2: Aprire la chiavetta USB

**Su Windows:** Potrebbe apparire una notifica che indica che e stato rilevato un nuovo dispositivo. Cliccaci per aprire la chiavetta. Se non appare nessuna notifica, apri **Esplora File** (icona della cartella nella barra delle applicazioni) e cerca la chiavetta USB sotto **Questo PC** sul lato sinistro. Avra un nome come "Unita USB (E:)" o il nome della marca.

**Su Mac:** La chiavetta USB apparira sulla **Scrivania** come un'icona, e anche nella barra laterale del **Finder** sotto **Posizioni**.

Fai doppio clic per aprirla e vedere cosa contiene.

### Passo 3: Copiare file sulla chiavetta USB

Per copiare un file dal computer alla chiavetta USB:

1. Apri la cartella dove si trova il file sul computer.
2. Clicca con il tasto destro sul file e seleziona **Copia** (o premi **Ctrl + C** su Windows, **Cmd + C** su Mac).
3. Apri la cartella della chiavetta USB.
4. Clicca con il tasto destro dentro la cartella della chiavetta e seleziona **Incolla** (o premi **Ctrl + V** su Windows, **Cmd + V** su Mac).

Il file verra copiato sulla chiavetta USB. Apparira una barra di avanzamento se il file e grande.

### Passo 4: Copiare file dalla chiavetta USB al computer

Il processo e lo stesso, ma al contrario:

1. Apri la cartella della chiavetta USB.
2. Clicca con il tasto destro sul file che vuoi e seleziona **Copia**.
3. Vai alla cartella del computer dove vuoi salvarlo (per esempio, Documenti).
4. Clicca con il tasto destro e seleziona **Incolla**.

### Passo 5: Rimuovere la chiavetta USB in sicurezza

Prima di togliere la chiavetta USB, espellila sempre in sicurezza:

**Su Windows:** Clicca sulla **piccola freccia** nell'angolo in basso a destra dello schermo (area di notifica). Clicca sull'**icona USB** e seleziona **Espelli** accanto al nome della chiavetta. Aspetta il messaggio "E possibile rimuovere l'hardware in sicurezza".

**Su Mac:** Clicca con il tasto destro sull'icona della chiavetta USB sulla Scrivania e seleziona **Espelli**. Oppure clicca sul **pulsante di espulsione** (triangolo con una linea) accanto al nome della chiavetta nel Finder.

### Passo 6: Togliere la chiavetta USB

Una volta vista la conferma che e sicuro, togli delicatamente la chiavetta USB dalla porta.

Conservala in un posto sicuro. Molte chiavette USB hanno un cappuccio o un connettore retrattile per proteggere la spina.

## Consigli

- Etichetta le tue chiavette USB o usa colori diversi per ricordare cosa c'e su ciascuna.
- Le chiavette USB sono ottime per condividere file con familiari e amici vicini. Copia i file e consegnala.
- Non usare una chiavetta USB come unica copia dei tuoi file importanti. Puo perdersi o danneggiarsi. Tieni sempre una copia sul computer o nel cloud.
- Se il tuo computer ha solo porte USB-C (piccole, di forma ovale), potresti aver bisogno di un piccolo adattatore per usare una chiavetta USB tradizionale.
