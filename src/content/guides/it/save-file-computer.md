## Cosa imparerai

In questa guida, imparerai a salvare file sul computer. Salvare garantisce che il tuo lavoro non vada perso quando chiudi un programma o spegni il computer.

## Cosa ti serve

- Un computer (Windows o Mac)
- Qualsiasi programma dove crei o modifichi file (editor di testo, elaboratore di testi, editor di foto, ecc.)

## Istruzioni passo per passo

### Passo 1: Capire cosa significa salvare

Quando lavori su un documento, una foto o qualsiasi file, le tue modifiche esistono solo nella memoria temporanea del computer. Se chiudi il programma o manca la corrente, quelle modifiche scompaiono.

Salvare scrive il tuo lavoro nell'archivio del computer (il disco rigido o SSD), dove resta in modo permanente finche non lo cancelli.

### Passo 2: Salvare per la prima volta

Quando crei un nuovo file, devi salvarlo per la prima volta.

Premi **Ctrl + S** (Windows) o **Cmd + S** (Mac). Apparira una finestra che ti chiede di:

1. **Scegliere un nome** per il file. Scegli qualcosa di descrittivo per trovarlo dopo (per esempio, "Lettera al dottore" invece di "Documento1").
2. **Scegliere una posizione** dove il file verra salvato. La cartella **Documenti** e una buona scelta.
3. Cliccare su **Salva**.

### Passo 3: Salvare le modifiche a un file esistente

Dopo il primo salvataggio, premi **Ctrl + S** (Windows) o **Cmd + S** (Mac) regolarmente mentre lavori. Questo aggiorna il file con le ultime modifiche.

Non c'e nessuna finestra questa volta. Il file viene salvato istantaneamente nella stessa posizione con lo stesso nome.

**Buona abitudine:** Salva ogni pochi minuti mentre lavori. Ci vuole meno di un secondo e ti protegge dalla perdita di lavoro.

### Passo 4: Usare Salva con nome per una copia

Se vuoi salvare una versione diversa di un file (per esempio, una traduzione o una versione aggiornata), usa **Salva con nome**:

Premi **Ctrl + Shift + S** (Windows) o **Cmd + Shift + S** (Mac), oppure vai su **File > Salva con nome** nel menu.

Dai un nuovo nome al file (per esempio, "Lettera al dottore - aggiornata") e scegli una posizione. Clicca su **Salva**.

Il file originale resta invariato. Ora hai due file.

### Passo 5: Trovare i file salvati

Per trovare i file che hai salvato:

**Su Windows:** Apri **Esplora File** (l'icona della cartella nella barra delle applicazioni). Clicca su **Documenti**, **Desktop** o la cartella dove hai salvato.

**Su Mac:** Apri il **Finder** (l'icona della faccina sorridente nel dock). Clicca su **Documenti**, **Scrivania** o la cartella pertinente.

Puoi anche usare la barra di ricerca. Premi il **tasto Windows** e digita il nome del file, oppure premi **Cmd + Spazio** su Mac e digita il nome.

## Consigli

- Dai nomi chiari ai tuoi file. "Budget marzo 2026" e molto meglio di "Senza titolo" o "Nuovo documento."
- Crea cartelle per organizzare file correlati. Clicca con il tasto destro in una cartella e seleziona **Nuovo > Cartella** (Windows) o **Nuova cartella** (Mac).
- Se un programma si blocca prima di salvare, cerca i file recuperati automaticamente. Molti programmi come Word e Google Docs salvano bozze automaticamente.
